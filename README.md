# 🎬 CineTrack

CineTrack is a social movie tracking app where you can discover films, keep a diary of what you've watched, and share recommendations with friends.

---

## ✨ Features

- **Movie Diary**: Keep track of every movie you've seen.
- **Watchlist & Likes**: Save movies to watch later or heart your favorites.
- **Smart Recs**: Get movie suggestions based on what you and the community like.
- **Social**: Join groups, chat with friends, and see what others are watching.
- **Reviews**: Rate movies and read reviews from the CineTrack community.

---

## 🛠️ Built With

- **Backend**: Node.js & Express
- **Database**: MongoDB
- **Data Source**: The Movie Database (TMDB) API

---

## 🚀 Quick Start

1. **Install**:
   ```bash
   npm install
   ```
2. **Setup**: Add your `MONGO_URI` and `TMDB_API_KEY` to a `.env` file.
3. **Run**:
   ```bash
   npm run dev
   ```
